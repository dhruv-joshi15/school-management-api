# School Management API

## 🚀 Project Overview
This is a RESTful API for managing schools, built using Node.js, Express, MongoDB, and Docker.

## 🛠️ Technologies Used
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- Docker
